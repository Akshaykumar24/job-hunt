# job-hunt

- Clone this Repo

## Front-End

- To install all dependecies run

``` npm install ```

- To Run the App

``` npm run start ```

## Back-End

- You'll have to add MongoAtlas connect link in ``` .env ``` file to access data
- To install all dependecies run

``` npm install ```

- To Run the App

``` npm run start ```

Backend will run on port 8080 by default, if you dont provide it in .env file
